# Rock, Paper, Scissors Game
This is a simple implementation of the classic game, Rock, Paper, Scissors, written in JavaScript. The game is played against a computer opponent.

## How to Play

1. Run the `code on the browser
2. You will be prompted to enter your choice ('rock', 'paper', or 'scissor').
3. open dev tool of your browser and go to the console tab


## Game Rules
- Rock beats Scissors: Scissors beat Paper
- Scissors beat Paper
- Paper beats Rock
- 
## Installation
This game is written in JavaScript and can be run in any JavaScript environment. No additional dependencies are required.
